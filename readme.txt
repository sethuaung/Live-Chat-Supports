Free Live Support Chat for your WordPress website.

== Description ==

LiveSupporti for WordPress allows you to add **free live support chat** to your WordPress website so you can chat with your customers and provide great customer support.

You can use **Forever Free Plan** with **3 agents**, **unlimited** websites, **unlimited** simultaneous chats, **unlimited** chat sessions, SSL encryption and chat history.




See the Demo <a href="http://livesupporti.com/live-chat-demo" target="_blank">here</a>.

Get LiveSupporti for **iOS** from <a href="https://itunes.apple.com/us/app/livesupporti/id957363647?mt=8" target="_blank">here</a>

Get LiveSupporti for **Android** from <a href="https://play.google.com/store/apps/details?id=com.livesupporti.android&hl=en" target="_blank">here</a>.


**How to install**

1. <a href="https://livesupporti.com/signup" target="_blank">Sign Up</a> for a free account.
2. Log into your WordPress admin panel, go to the **Plugins** menu, click **Add New**, search for **livesupporti** plugin and install it.
3. Click on **LiveSupporti** menu on the left and enter your LiveSupporti credentials. You're done!

== Installation ==

1. Create your free account at <a href="https://livesupporti.com/signup" target="_blank">livesupporti.com/signup</a>
2. Log into your WordPress admin panel and go to the **Plugins** menu.
3. Click **Add New**, enter **livesupporti** and click **Search Plugins**.
4. Click **Install Now**, and then **Activate Plugin**.
5. Enter your LiveSupporti credentials and click **Save**.
